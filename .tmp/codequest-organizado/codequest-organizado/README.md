# CodeQuest - Projeto organizado

Este pacote foi separado em áreas para ficar mais fácil de editar e evoluir.

## Arquivos principais

- `index.html` — estrutura da plataforma e páginas.
- `styles.css` — visual, layout, cores, cards, sidebar e responsividade base.
- `script.js` — navegação entre páginas, botões de filtro, abas e painel lateral.
- `assets/` — pasta pronta para colocar imagens, ícones e ilustrações.
- `docs/` — pasta para anotações, ideias e documentação.
- `src/` — cópia organizada dos arquivos principais, caso queira migrar depois para React/Vite.

## Como abrir

1. Extraia o `.zip`.
2. Abra o arquivo `index.html` no navegador.

## Próximos upgrades recomendados

- Colocar imagens ilustrativas em `assets/`.
- Separar dados das trilhas em um arquivo JSON.
- Migrar para React quando for transformar em app real.
- Conectar autenticação com Supabase depois que a interface estiver estável.
